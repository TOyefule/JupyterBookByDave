What is JavaScript?

Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web.. JavaScript is the programming language of the browser. It allows you to make web pages interactive by manipulating the HTML Document (called the DOM) that is loaded in the browser. Each web browser implements its own JavaScript engine so JavaScript code runs on the client, not the server. This relieves server of processing effort for simple tasks like validating data and gives the user a more interactive web experience.

What is the DOM?

The DOM (Document Object Model) is the collection of objects that make up a web page. These objects are structured as a collection of nodes - each connected to another.

What is event-driven Programming?

Event-driven programming is a programming paradigm in which the flow of the program is determined by events such as user actions, sensor outputs, or messages from other programs/threads.

When does JavaScript execute?

JavaScript code will execute in response to an event being fired. So if a user clicks a button or a web page finishes loading an event will be fired. In the first case, the button_click event and the on_load event. These events are set (or registered) as event handlers on the nodes of the DOM.
