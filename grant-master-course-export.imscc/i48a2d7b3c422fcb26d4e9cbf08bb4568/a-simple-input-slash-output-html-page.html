Let's create a web page that allows us to use JavaScript to input something, such as our name, and will display a result on the web page. This will allow us to play with JavaScript in future examples. Simply copy the following code to a static web application and run it in the browser.

 

<!DOCTYPE html>
<html>
<body>

<h2>JavaScript Input and Output</h2>
<p id="output"></p>
<script>
var name = prompt("Please enter your name", "<name goes here>");

if (name!= null) {
document.getElementById("output").innerHTML =
"Hello " + name + "! How are you today?";

}
</script>


</body>
</html>
