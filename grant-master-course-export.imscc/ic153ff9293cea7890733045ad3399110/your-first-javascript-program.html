Create your first script

You can add this script to your crab cake recipe site.

Create a script that simply says Hello, World! (or something of your choosing) when the user clicks a button.

Your script looks like this and can go in between the <head> and </head> tags of your web page.

The script starts with an html comment to hide it from browsers with JavaScript disabled.

<script language="JavaScript">
<!--
function Hello()
 {
 alert("Hello World!")
 }//-->
</script>

 

Place the script between the <head> tags of your web page and modify your body tag as follows to call the function when the page loads. Then, save your project and refresh your page.

<body onload="Hello()">
