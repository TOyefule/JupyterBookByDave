Add the following button and the following script to your web page that already contains a table. Place the script just before the closing body tag. Then when the user clicks the button it will add another row to the table. 

Can you make it so it adds a row to the end?

Can you pre-populate the values with text?

 

<button onclick="myFunction()">Add Rows</button>
<script>
function myFunction() {
 var table = document.getElementById("myTable");
 var row = table.insertRow(0);
 var cell1 = row.insertCell(0);
 var cell2 = row.insertCell(1);
 cell1.innerHTML = "NEW CELL1";
 cell2.innerHTML = "NEW CELL2";
}
</script>
