You can make your program more efficient and write less code if you use loops to repeat blocks of code. You can even include conditionals inside the loop to determine when to exit the loop. Loops allow your program to perform the same task over and over again enabling it to rapidly complete repetitive tasks.

Loops always need an exit condition. Many loops implement some sort of counter while others require a variable or condition to be checked each time through the loop to decide if the loop should continue running or exit.
