HTML is used to specify whether your web content should be recognized as a paragraph, list, heading, link, image, multimedia player, form, or one of many other available elements or even a new element that you define.

 

The following is a valid html page. You can check it at https://validator.w3.org/nu/#textarea

<!DOCTYPE html>
<html lang="en" xml:lang="en" xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta charset="utf-8">
<title>My test page</title>
</head>
<body>

<h1>My First Heading</h1>
<h2>My sub heading</h2>

An unordered list:
<ul>
<li>One fish</li>
<li>two fish</li>
<li>red fish</li>
<li>blue fish</li>
</ul>

<p>My first paragraph.</p>
<p>My second paragraph.</p>
