Conditional execution of code allows you to write a program which contains code that will only run under certain situtations.

Most programming languages use the if..else construct to implement decision making. You can implement if ... else in JavaScript as follows:

var shoppingDone = true;

if (shoppingDone) { // don't need to explicitly specify '=== true'
  var childsAllowance = 10;
} else {
  var childsAllowance = 5;
}
